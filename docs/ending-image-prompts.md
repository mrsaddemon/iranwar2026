# Ending Image Prompt Pack

These are rebuilt to create stronger separation across both:
- **country identity**
- **outcome identity**

Shared style suffix:

`vertical cinematic ending-card illustration, grounded geopolitical realism, dramatic but tasteful lighting, dark war-room palette, readable at thumbnail size, strong focal subject, subtle environmental storytelling, no text, no UI, no watermark, polished concept art, high detail`

## Visual differentiation rules

### Country identity
- **USA**: maritime power, carrier decks, naval command glass, ISR aircraft, strategic sea-lane control
- **Israel**: urban skyline, missile-defense arcs, rooftop or command-over-city angle, compressed frontline tension
- **Iran**: mountain-backed skyline, Tehran command core, missile infrastructure, land-based strategic pressure

### Outcome identity
- **Win**: elevated camera, clean horizon, stronger central icon, disciplined geometry
- **Draw / Ceasefire**: balanced but uneasy composition, split focal zones, dusk or haze, unresolved motion
- **Loss**: tilted composition, damage, fatigue, fragmented lighting, visible cost
- **Catastrophic**: blackout, smoke, emergency strobes, towering destruction, oppressive vertical scale
