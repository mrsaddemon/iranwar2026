- `usa-draw.png`

- `usa-catastrophic.png`
- `israel-win.png`
- `israel-draw.png`

- `iran-win.png`

Or nested paths:

- `usa/win.png`
- `usa/draw.png`
- `usa/loss.png`
- `usa/catastrophic.png`
- `israel/win.png`
- `israel/draw.png`
- `israel/loss.png`
- `israel/catastrophic.png`
- `iran/win.png`
- `iran/draw.png`

Supported file types:

- `.png`
- `.jpg`
- `.jpeg`
- `.webp`

The app will automatically pick up the first matching asset for the final personalized conclusion card.
